# Product Knowledge Gap Analysis

## Workflow

1. Inspect product attributes, variants, specs, ingredients/materials, sizing, compatibility, use cases, and contraindications.
2. Check whether product pages answer buyer questions without support escalation.
3. Identify missing schema or feed fields.
4. Group gaps by template-level fixes vs product-specific enrichment.
5. Recommend the smallest product knowledge load needed for a custom agent.

## Output

- Missing attribute or answer
- Affected product/template
- Customer or AI-agent failure mode
- Suggested source of truth
- Remediation priority
